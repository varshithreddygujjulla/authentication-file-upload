# Authentication System with File Upload 🔐

This project is a backend authentication system that allows users to register, log in, and upload files securely.

## Features
- User registration
- User login authentication
- Secure password hashing using bcrypt
- File upload functionality
- Store user data in MongoDB

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer
- bcrypt

## Installation

1. Clone the repository

git clone https://github.com/varshithreddygujjulla/authentication-file-upload.git

2. Install dependencies

npm install

3. Start the server

node app.js

4. Open in browser

http://localhost:8080
